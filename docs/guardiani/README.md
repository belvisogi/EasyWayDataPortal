# Guardiani del Codice - Docs

Questa cartella contiene materiale riusabile per i "Guardiani del Codice": template e esempi (redatti) di output.

Obiettivo
- Separare il documento canonico (concetto e regole) dagli esempi operativi.
- Evitare di versionare nel repo dettagli sensibili (IP, username, key name, segreti, comandi con credenziali).

Struttura
- `docs/guardiani/examples/`: esempi redatti (sicuri da condividere/indicizzare)

Nota privacy
- Se hai report reali con IP pubblici, credenziali, nomi di chiavi SSH, variabili d'ambiente con segreti: NON linkarli in wiki/KB e non includerli in indici LLM.
- Usa gli esempi redatti come formato standard e allega i dettagli sensibili solo fuori repo.
